# Rust Experiments

A personal sandbox repository for exploring Rust — language features, standard library patterns, and anything worth poking at.

## Structure

```
src/
  main.rs   # Entry point; experiments go here or in separate modules
```

## Running

```bash
cargo run
```

## Testing

```bash
cargo test
```

## Notes

- Each experiment is self-contained and may overwrite `main.rs` or add new modules
- No production code — purely exploratory
